# Do you have meetings outside your working hours?

Ever wondered how many of your meetings actually happen outside your normal working hours? This little script will tell you.

## How to use it

1. Clone this project.
2. Run `npm install`.
3. Export your calendar:
   - In Google Calendar: go to **Settings → Import & export → Export** and download your calendar.
4. Open `index.mjs` and configure:
   - Your email address
   - The path to your `.ics` file
   - Your working hours
   - Your time zone
5. Run:

   ```bash
   node index.mjs
   ```


Warning: The script is generated by ChatGPT (and tuned by me)
